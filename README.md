# song_reco
介绍：Scrapy框架爬取网易云音乐热歌榜数据，存储到MySQL数据库，并对数据做可视化分析。
    1.创建数据库
    2.Scrapy框架爬取网易云音乐的热歌榜前200首歌曲（歌曲风格随机选择），并存储到mysql数据库
    3.matplotlib数据可视化
        3.1画出歌手与歌曲排名的散点图；
        3.2各种风格歌曲的数量（柱状图）；
        3.3不同歌曲时长的比例（饼图）
涉及的知识点：selenium、Scrapy框架、matplotlib数据可视化。

拓展：
    1.下载文件（歌曲和歌手头像）
    2.使用正则表达式匹配组合歌手的第一位歌手，即’/‘之前的字符串；或者使用字符串的split方法实现
